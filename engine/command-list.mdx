## Miscellanous
Uncategorised or globally available commands.

### `/config`
<Card>
  <Note>You must be an **Administrator** or the **Server Owner** to use this command.</Note>
  Allows you to configure and set up Compass in the current server.
  <ResponseField name="config" type="Application Command"/>
</Card>

### `/ping`
<Card>
  <Check>This command is **publically available**.</Check>
  Replies with pong! See the latency of the bot.
  <ResponseField name="ping" type="Application Command" />
</Card>

### `/help`
<Card>
  <Check>This command is **publically available**.</Check>
  See important information about the bot, join the support server, and view all commands.
  <ResponseField name="help" type="Application Command" />
</Card>

### `/botstats`
<Card>
  <Check>This command is **publically available**.</Check>

### `/erlc players`

  <ResponseField name="erlc info" type="Application Command" />

## Giveaways
Commands to create and manage giveaways.

### `/giveaway create`
<Card>
  Creates a giveaway with a duration, number of winners, and prize.
  <ResponseField name="giveaway create" type="Application Command">
    <Expandable title="options">
      <ParamField path="prize" type="string" required>
        The prize for the giveaway.
      </ParamField>
      <ParamField path="input" type="number" required>
        The number of winners.
      </ParamField>
      <ParamField path="duration" type="Duration" required>
        The duration of the giveaway. (e.g. `1h`, `30m`)
      </ParamField>
    </Expandable>
  </ResponseField>
</Card>


### `/giveaway re-roll`
<Card>
  Re-roll the winner of an ended giveaway via it's ID.
  <ResponseField name="giveaway re-roll" type="Application Command">
    <Expandable title="options">
      <ParamField path="id" type="id" required>
        The command you want to run.
      </ParamField>
    </Expandable>
  </ResponseField>
</Card>
